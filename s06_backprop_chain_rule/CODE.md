# s06 反向传播与链式法则 -- 代码说明与运行报告

## 程序做了什么
从零实现一个微型自动微分引擎（mini autograd），模拟PyTorch的`.backward()`核心逻辑。每个`Value`节点记录data、grad、_backward和_prev，通过拓扑排序按逆序执行链式法则。包含5个演示：基本表达式求导、激活函数反向传播、Fan-out梯度累积、小神经网络训练、以及梯度下降求函数最小值。

## 运行方法
```bash
cd s06_backprop_chain_rule/code
python demo.py
```

## 运行结果

### 输出摘要
全部为控制台文本输出，不生成图片。

**演示1 -- 基本表达式 (a*b + c) * d：**
- 输入: a=2, b=3, c=4, d=5, 结果 L=50
- 梯度验证: dL/da=15, dL/db=10, dL/dc=5, dL/dd=10 -- 全部与手动推导一致

**演示2 -- 激活函数反向传播：**
- ReLU(1.5): 导数=1.0（正区梯度通过）, ReLU(-1.5): 导数=0.0（负区梯度截断）
- Sigmoid(1.5): 导数与 σ(x)(1-σ(x)) 公式一致
- Tanh(1.5): 导数与 1-tanh^2(x) 公式一致

**演示3 -- Fan-out 梯度累积：**
- 表达式 L = (2x)*(x+3), x=2 时 dL/dx=14
- 梯度分解: 路径1贡献=10（通过乘法分支），路径2贡献=4（通过加法分支），总和=14
- 这展示了多元链式法则中多条路径的梯度自动累加

**演示4 -- 小神经网络训练 (2->4->1)：**
- 4个训练样本，目标函数 y = 3x1^2 - 2x2 + 1
- 训练100轮后MSE损失下降，预测值逐步逼近真实值

**演示5 -- 梯度下降求 f(x)=x^2+3x 最小值：**
- 从x=5出发，经20步梯度下降
- 最终收敛到 x ≈ -1.5（与解析解一致），f_min ≈ -2.25

### 计算图结构
每次演示后打印计算图的拓扑结构：节点按深度排序，展示data值、累积的grad值、操作类型和父子关系。例如演示1中打印了7个节点的完整计算图。

## 代码结构
- `class Value` -- 自动微分的核心节点，支持 `+`, `*`, `**`, `/`, `-` 等算术运算以及 `relu()`, `sigmoid()`, `tanh()`, `exp()` 激活函数；`backward()` 执行拓扑逆序梯度传播
- `class Neuron` / `class Layer` / `class MLP` -- 使用Value构建的简单神经网络
- `trace_graph()` / `print_computation_graph()` -- 计算图结构与梯度信息的文本打印
- `demo_basic_expression()` -- 演示1：基本表达式链式法则
- `demo_activation_functions()` -- 演示2：激活函数反向传播
- `demo_fanout()` -- 演示3：梯度累积
- `demo_mini_neural_network()` -- 演示4：完整MLP训练流程
- `demo_gradient_descent_1d()` -- 演示5：梯度下降求函数极值
